# 🎬 Netflix Movie Recommendation System

## Project Overview

This project develops a Content-Based Movie Recommendation System using the Netflix Movies & TV Shows dataset. It recommends similar movies and TV shows based on genres, descriptions, directors, and cast information using TF-IDF Vectorization and Cosine Similarity.

---

## Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- TF-IDF Vectorization
- Cosine Similarity
- Content-Based Recommendation System
- Personalized Movie Recommendations

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

---

## Dataset

Netflix Movies & TV Shows Dataset

Columns used:

- Title
- Type
- Director
- Cast
- Country
- Release Year
- Rating
- Listed In
- Description

---

## Exploratory Data Analysis

The project includes:

- Movies vs TV Shows Distribution
- Top Genres
- Ratings Distribution
- Release Year Analysis
- Country-wise Content
- Movie Duration Analysis

---

## Machine Learning

Recommendation Technique:

- TF-IDF Vectorization
- Cosine Similarity

The system recommends the Top 5 similar movies based on content features.
